# reef
Carbon emission calculator

This app relies on [Git](https://git-scm.com/).  Download it if you don't have it.

## Getting Started

Get the project files.

1. Open VS Code
2. Open Quick Access in top bar
3. Run command `> Git: clone`
4. Select the `reef` repository on Github

Setup the project.

1. Open `Extensions` Panel in VS Code
2. Search `@recommended`
3. Install the recommended extensions.
4. Select `Run > Start Debugging` in the menu bar. `F5` shortcut.
